## Por qué priorizar clústeres sobre tests aislados

En el hombro, la mayoría de los tests especiales aislados tienen precisión limitada: alta sensibilidad con baja especificidad (Neer, Hawkins-Kennedy) o alta especificidad con baja sensibilidad (drop-arm, lift-off). La combinación de tests (clústeres) casi siempre mejora el valor diagnóstico frente a cualquier test suelto, porque suma la capacidad de descarte de los tests sensibles con la de confirmación de los específicos. La regla general es: usar tests sensibles para cribar y tests específicos (o clústeres) para confirmar, e informar el rendimiento según el NÚMERO de ítems positivos.

## Clúster de Park para rotura transmural del manguito rotador

Combina tres hallazgos para la rotura de espesor completo del manguito:
1. Arco doloroso.
2. Signo del brazo que cae (drop-arm).
3. Debilidad en rotación lateral resistida (infraespinoso).

Rendimiento diagnóstico (Park et al.):
- Con los tres tests positivos, el LR+ para rotura transmural del manguito es muy alto (del orden de 15), con una probabilidad postest elevada.
- Para el impingement/bursitis subacromial, la combinación de Hawkins-Kennedy + arco doloroso + debilidad del infraespinoso también aporta un LR+ elevado (del orden de 10).

Interpretación clínica: es el clúster CONFIRMATORIO de referencia para rotura del manguito. Cuantos más ítems positivos, mayor la probabilidad de rotura transmural. Aplicable especialmente en pacientes mayores de 60 años con debilidad en rotación lateral y dolor nocturno.

## Clúster de Chronopoulos para patología acromioclavicular

Combina tres tests dirigidos a la articulación acromioclavicular:
1. Aducción horizontal cruzada (cross-body adduction).
2. Extensión resistida acromioclavicular.
3. Test de compresión activa de O'Brien (dolor localizado sobre la AC).

Rendimiento diagnóstico: con los tres positivos, el eAppendix de Magee recoge legible LR+ ~8.3, LR− ~0.77. Interpretación clínica: aproximación CONFIRMATORIA de patología AC; su valor está en la concordancia de los tres tests, no en cualquiera aislado.

## Clúster de Bankart / lesión labral anterior

Combina tres tests para la lesión labral anterior (Bankart):
1. Test de crank (manivela).
2. Test de aprensión anterior.
3. Test de recolocación de Jobe.

Rendimiento diagnóstico: el eAppendix de Magee recoge, de forma legible, LR+ ~6.0 con LR− ~0.12 para la combinación. Interpretación clínica: clúster equilibrado que sirve tanto para CONFIRMAR (LR+ 6.0) como para DESCARTAR (LR− 0.12) la lesión labral anterior; supera a los tests labrales individuales, cuya precisión aislada es limitada.

## Combinación para impingement subacromial (por número de tests positivos)

Magee recoge una combinación de seis tests para el impingement subacromial: Yocum, aducción horizontal, arco doloroso, drop-arm, Yergason y Speed. El rendimiento diagnóstico mejora con el número de tests positivos: cuantos más tests de la batería resultan positivos, mayor es la probabilidad de síndrome subacromial.

Interpretación clínica: útil como estrategia escalonada; un único test positivo aporta poca certeza, mientras que la positividad de varios tests de la batería aumenta progresivamente la probabilidad postest. Prioriza siempre esta lógica de recuento sobre la interpretación de un test aislado.

## Tríada de inestabilidad anterior (aprensión + recolocación + liberación)

Aunque se describe en el archivo de tests de inestabilidad, funciona como clúster: la secuencia aprensión → recolocación → liberación/"sorpresa" es la aproximación de referencia a la inestabilidad anterior. El test de liberación es el de mayor especificidad (LR+ ~8.3, LR− bajo). La concordancia de los tres, usando la APRENSIÓN (no el dolor) como criterio, ofrece el mayor valor confirmatorio para inestabilidad anterior.
